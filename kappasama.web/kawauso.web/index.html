カワウソについて

カワウソとは..
カワウソは、ネコ目イタチ科カワウソ亜科に属する肉食動物。
南極・オーストラリア・ニュージーランドを除く世界全域に生息している7属13種の総称です（ラッコもカワウソの仲間です）。
主に河川や湖沼、海岸などの水辺に生息していますが、地域によっては水田など人の生活に近いところに棲んでいることもあります
