KAPPA COOKIEかっぱクッキーのこと

たいせつなこと

私たちが大切にしていることはお客さまの“安全”です。 安心して食べることができるように衛星管理を徹底しております。また、食品においても産地などにこだわりをもって仕入れております。

これからのこと

お子さまから大人の方まで皆さんが美味しく食べられるクッキーをこれからも作ってまいります。見た目だけではなく味にもこだわっております。ぜひご賞味ください♪

貧困問題について

子ども食堂への提供、イベントの開催

クッキー専門店ではありますが、私達なりにできることはあるのではないかと考え、店内でのお飲み物の提供、子どもの貧困問題に向き合うため、こども食堂へのおやつの提供、夏休みなどこどもの長期休み期間に親子向けのワークショップ『親子手作りクッキー教室』(無料実施)を致しております。

差別のない社会を目指して

ジェンダー問題について

現在、ジェンダー問題について認知されてまいりましたが、まだまだ差別はあると感じております。 ジェンダーレス社会に向けて店内のイベントなども男女分け隔てなく参加できるように考えております。 母子だけでなく父子でも気兼ねなく過ごせるお店にしていけるよう目指してまいります。

男女共同参画社会に向けて

また、地域の福祉の方と連携しているため男女参画、家庭の悩みなどの相談ができるよう場所の提供をいたしております。月2回ほどNPO法人の方が相談会を開いております。

環境問題について

プラスチック製品の削減

海に溶け込むプラスチック製品が問題になっておりますが、私達にも少しでも出来ることがないかと考え、このお店では“バイオマスストロー”を使用しております。植物などに由来する再生可能な有機性資源（バイオマス）を主原料とするプラスチック製のストローになります。
